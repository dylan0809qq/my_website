## 本地測試

1. 確保您有 Python 或 Node.js 安裝
2. 在專案目錄中運行：
   # 使用 Python
   python -m http.server 8000
   
   # 或使用 Node.js
   npx http-server . -p 8000
   `
3. 在瀏覽器中訪問：http://localhost:8000

## 部署到 GitHub Pages

1. 創建一個新的 GitHub 倉庫
2. 將所有文件上傳到倉庫
3. 在倉庫設定中啟用 GitHub Pages
4. 選擇 main 分支作為來源
